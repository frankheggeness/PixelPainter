Welcome to Pixel Painter!

Clear: Click once to clear entire canvas.

Erase: Click once to set paint color to white.

Randomize One Color: Click button once, then click on any color in the random colors swathe to randomize it.

Randomize All: Click once to randomize all colors in the random colors swathe.

Set Favorite Color: First, click on color you wish to save. Afterward, click on the "Set Favorite Color" button. Finally, click any square in the favorite color swathe to set your color.

Save Canvas: Click once to create code in the input box below. Copy this code to your clipboard to load your canvas later.
